# 歧视诉讼中的回归分析：如何读懂和挑战统计证据

> **涉及知识域**：多元统计、统计分析

## 案情

Jane Reno 是 ISA（Ivy Student Agencies）的一名女性员工，因被拒绝晋升而咨询你是否应提起性别歧视投诉。

**基本数据**：女性员工平均年薪 $8,763，男性平均年薪 $9,620。

ISA 是一家校园公司，6 年前由 3 名男性本科生创立，现已扩展至洗衣、清洁、电器租赁、辅导和旅游指南出版。员工来自大学生和当地高中生。三位创始人从法学院毕业后留任管理层。

**回归分析结果**（OLS，n=125，因变量 = 年薪）：

| 变量 | 系数 | 标准误 |
|------|------|--------|
| 常数项 | 4556.7 | 498.6 |
| 性别 (Sex) | 903.16 | 174.2 |
| 教育 (Education) | 70.40 | 150.7 |
| 工龄 (Months) | 75.40 | 38.31 |
| 年龄 (Age) | 55.83 | 112.48 |

R² = 0.603

> 性别编码：男 = 1，女 = 0。教育：超过八年级的年数。工龄：在 ISA 工作的月数。年龄：实际年龄 - 14。

---

## 分析

### 一、薪酬预测

**人物画像**：23 岁男性，研究生一年级，在 ISA 工作 26 个月。

**变量编码**：
- Education = 4（高中）+ 4（本科）+ 1（研一）= **9**
- Months = **26**
- Age = 23 - 14 = **9**

**男性预测薪酬**（Sex = 1）：

$$4556.7 + 903.16 \times 1 + 70.40 \times 9 + 75.40 \times 26 + 55.83 \times 9$$
$$= 4556.70 + 903.16 + 633.60 + 1960.40 + 502.47 = \textbf{\$8,556.33}$$

**女性预测薪酬**（Sex = 0）：

$$4556.70 + 0 + 633.60 + 1960.40 + 502.47 = \textbf{\$7,653.17}$$

**性别差距 = $903.16** —— 恰好等于性别系数。在教育、工龄、年龄完全相同的情况下，男性每年多赚 $903。

### 二、系数显著性检验

t 统计量 = 系数 / 标准误。自由度 = 125 - 5 = 120，双尾检验 .05 水平的临界值约为 **1.98**。

| 变量 | 系数 | 标准误 | t 统计量 | .05 显著？ |
|------|------|--------|---------|-----------|
| 常数项 | 4556.7 | 498.6 | **9.14** | 是 |
| 性别 | 903.16 | 174.2 | **5.18** | **是 ★★★** |
| 教育 | 70.40 | 150.7 | **0.47** | 否 |
| 工龄 | 75.40 | 38.31 | **1.97** | 否（极度边界） |
| 年龄 | 55.83 | 112.48 | **0.50** | 否 |

**逐项解读**：

- **性别**（t = 5.18）：**高度显著**。控制教育、工龄、年龄后，性别仍然显著影响薪酬。这是 Reno 歧视主张的核心统计证据。

- **教育**（t = 0.47）：**不显著**。ISA 员工主要是大学生和高中生，教育年限变异范围很窄，且 ISA 的工作（洗衣、清洁、辅导）可能本来就不特别奖励额外学历。

- **工龄**（t = 1.97）：**极度接近但未达 .05 临界值**（1.97 < 1.98）。实际上处于边界。更大样本或单尾检验即可显著。

- **年龄**（t = 0.50）：**不显著**。很可能因为**多重共线性** —— 年龄与教育和工龄高度相关。当三者同时进入模型，年龄的独立解释力被消耗殆尽。加之学生群体的年龄范围本就很窄。

### 三、为什么回归优于简单均值比较

原始薪酬差距：$9,620 - $8,763 = **$857/年**。

| 方法 | 局限 |
|------|------|
| 简单均值差异 | 混淆了所有因素（经验、学历、工龄、歧视），被告可以辩称"男性薪酬高是因为工龄更长" |
| 多元回归 | **控制了合法因素后隔离性别的纯效应**。控制后差距反而从 $857 扩大到 $903，说明女性在其他变量上可能略有优势，简单均值反而低估了歧视程度 |

回归还提供了 **t 统计量和 p 值**，可以正式检验性别差距是否由偶然产生（p < 0.001，几乎不可能）。

### 四、替代解释

尽管回归结果很强，但存在一个合理的替代解释：

ISA 由 **3 名男性** 创立，他们作为创始人兼管理者，很可能是薪酬最高的员工。他们的高薪可能拉高了性别系数 —— 不是因为 ISA 歧视女性，而是因为创始团队恰好全是男性。

**应对策略**：
- 排除 3 名创始人后重新回归
- 或增加"管理层/创始人"虚拟变量

此外，R² = 0.603 意味着 **40% 的薪酬变异未被解释**。遗漏变量（岗位类型、职级、绩效、工时）可能解释部分差距。**遗漏变量偏误** 是歧视案件中被告的经典辩护策略。

### 五、管理层性别比例检验

**额外数据**：130 名管理者中 44% 是女性，而员工中 56% 是女性（当地人口中 52% 是女性）。

#### 与员工群体比较（更强的论证）

零假设：管理者从 56% 女性的员工池中随机产生。

$$z = \frac{0.44 - 0.56}{\sqrt{0.56 \times 0.44 / 130}} = \frac{-0.12}{0.0435} = \textbf{-2.76}$$

|z| = 2.76 > 1.96 → **.05 水平下统计显著**（甚至在 .01 水平下也显著）。

#### 与当地人口比较（较弱）

$$z = \frac{0.44 - 0.52}{\sqrt{0.52 \times 0.48 / 130}} = \frac{-0.08}{0.0438} = \textbf{-1.83}$$

|z| = 1.83 < 1.96 → **.05 水平下不显著**。

员工群体是更合理的比较基准，因为管理者是从现有员工中晋升的。

## 总结

Jane Reno 有两条相互补充的统计证据：

| 证据 | 结果 | 显著性 |
|------|------|--------|
| 薪酬回归 | 控制教育、工龄、年龄后，男性年薪多 $903 | t = 5.18，p < 0.001 |
| 管理层比例 | 女性占员工 56% 但仅占管理层 44% | z = 2.76，p < 0.01 |

两者共同构成了薪酬歧视和晋升歧视的初步证据（prima facie case）。被告的可能抗辩方向：遗漏变量（岗位类型、创始人特殊地位）。案件的最终强度取决于加入这些控制变量后，性别效应是否仍然显著。

**核心教训**：简单的均值比较永远不够。多元回归通过控制合法因素来隔离歧视效应 —— 但回归本身也可以被挑战：遗漏变量、多重共线性、样本选择，每一个都是律师可以追问的方向。
