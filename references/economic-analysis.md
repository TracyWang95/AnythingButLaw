# 法经济学 Economic Analysis of Law

> 用经济学思维分析法律制度本身：为什么合同法要这样设计？和解什么时候比审判更好？刑罚的最优力度是多少？法经济学把法律制度视为一套激励系统来研究。

## 合同分析

### 基本概念
- **合同**：对具名各方在不同时间点、不同条件下应采取行动的约定
- **完全合同 (completely specified contract)**：包含所有可能条件下的行动规定 — 现实中几乎不存在
- **帕累托最优 (Pareto optimal)**：各方无法通过修改合同来提高任何一方的预期福利而不损害其他方

### 合同不完全性
各方通常 **希望** 不完全合同被解释为能最大化预期福利。这为法院的合同解释和缺省规则 (default rules) 提供了经济学依据。

### 违约损害赔偿

| 类型 | 含义 | 效果 |
|------|------|------|
| **期望损害赔偿 (Expectation Damages)** | 使受害方处于合同被履行时的状态 | 效率违约 (efficient breach) 的理论基础 |
| **信赖损害赔偿 (Reliance Damages)** | 补偿受害方因信赖合同而发生的支出 | 保守但可能不足 |
| **恢复原状 (Restitution)** | 返还违约方获得的利益 | 最低限度 |
| **约定损害赔偿 (Liquidated Damages)** | 双方预先约定的金额 | 需合理，否则被视为罚金而不可执行 |

### 实际履行 Specific Performance
强制交付货物或履行服务。

- 效果类似于极高的损害赔偿
- 创造了履行的强激励
- **问题**：在效率上不应履行时仍强制履行；服务合同比财产合同更难执行

### 重新谈判 Renegotiation
- 当双方能找到彼此且了解合同细节时，重新谈判成本较低
- **优势**：只在帕累托最优时才会导致实际履行
- **局限**：无法解决不合理的风险分担；可能陷入谈判僵局

## 合同的法律推翻 Legal Overriding

法律在什么情况下应该推翻当事人的意思自治？

### 正当理由
- 防止对第三方的有害外部性
- 保护合同当事人免受福利损失（未成年人、无行为能力人）
- **不可让渡之物 (inalienable)**：人体器官、婴儿、投票权

### 法律外的执行机制
- 私人仲裁、选定的裁判者
- **声誉 (reputation)**：长期关系中的自我执行

## 民事诉讼的经济学

### 起诉决策
当事人会一直投入诉讼成本，只要成本低于预期收益。

### 和解 vs 审判

**和解增加的条件**：
- 双方对案件结果有相似的判断
- 风险规避 → 更倾向和解

**审判发生的条件**：
- 原告对胜诉概率的判断显著高于被告认为的败诉概率
- 潜在赔偿金额越大 → 判断分歧的影响越大 → 越难和解

### Priest-Klein 假说
进入审判的案件 **不代表** 争议的整体分布。大多数案件会和解，进入审判的是那些双方分歧最大的案件。这意味着审判结果的胜诉率接近50%不是巧合 — 是选择效应。

## 侵权法的经济学

### 严格责任 vs 过错责任

| | 严格责任 (Strict Liability) | 过错责任 (Fault-Based) |
|---|---|---|
| **何时有效** | 金钱制裁、施加成本低 | 需要判断过错标准 |
| **优势** | 执行简单 | 当制裁是监禁（社会成本高）时有根本优势 |
| **威慑效果** | 两者在理想条件下都可实现无社会成本的威慑 | |

## 刑法的经济学

### 刑罚的目的
通过制裁将加害人从可能造成伤害的位置上移除。

### 监禁 Incapacitation
- 吊销执照、禁止担任高管/董事
- 只有在释放后预期伤害超过监禁成本时才应监禁

### 为什么关注意图 (Intent)？
对有意加害者施加更高制裁，因为他们：
- 更难威慑（从行为中获益更大）
- 更可能隐藏行为
- 更可能造成更大伤害
- 更有可能重犯

### 未遂犯罪 Punishment of Attempts
惩罚未遂 → 提高不良行为被制裁的概率 → 增强威慑效果

### 不一致性 Incongruities
- 某些行为（如逃税）制裁不足
- 某些行为制裁过度
- **道德应得 (moral deserts)**：对决定最优惩罚水平的影响
