# 会计 Accounting

> 律师不需要做账，但需要读懂账。在尽调中发现问题、在诉讼中质疑财务数据、在谈判中判断对方的财务状况 — 这些都需要会计基础。

## 三大财务报表

### 1. 资产负债表 Balance Sheet (Statement of Financial Position)

某一时刻的财务快照。通常包含两个时间点（期初和期末）以便比较。

**基本等式**：
```
资产 (Assets) = 负债 (Liabilities) + 所有者权益 (Equity)
```

等价形式：
```
权益 = 资产 - 负债
```

**关键局限**：
- 某些重要资产不在表上（如品牌价值、人才、未确认的知识产权）
- 历史成本可能与当前价值严重脱节

### 2. 利润表 Income Statement

一段时期内的经营成果。

```
销售收入 (Sales/Revenue)
- 销售成本 (COGS)
= 毛利 (Gross Margin/Profit)
- 营业费用 (Operating Expenses)
= 营业利润 (Operating Income)
- 非营业项目 (Non-operating Items)
= 税前利润 (Income Before Taxes)
- 所得税 (Income Taxes)
= 净利润 (Net Income)
```

### 3. 现金流量表 Statement of Cash Flows

反映一段时期内现金的来源和去向，揭示企业的 **流动性 (liquidity)** — 按时支付款项的能力。

三大类别：
- **经营活动 (Operating)**：日常业务产生/消耗的现金
- **投资活动 (Investing)**：购买/出售长期资产
- **筹资活动 (Financing)**：借贷、发行股票、分红

**律师注意**：净利润 ≠ 现金流。一家公司可以"盈利"但现金为负（因为收入是应计的但钱还没收到）。

## 复式记账法 Double-Entry Bookkeeping

每笔交易同时影响两个账户：

| | 增加 | 减少 |
|---|---|---|
| 资产 | 借方 (Debit) | 贷方 (Credit) |
| 负债/权益 | 贷方 (Credit) | 借方 (Debit) |

**流程**：
1. 在日记账 (journal) 中记录交易
2. 过账到分类账 (ledger)
3. 期末关闭收入和费用账户
4. 转入留存收益 (retained earnings)
5. 编制财务报表

## 核心会计原则

### 历史成本 Historic Cost
以购买时的实际支付金额记录。可靠 (reliable) 但不一定相关 (relevant) — 10年前花100万买的楼，现在可能值1000万。

### 配比原则 Matching Principle
收入归属于为产生该收入而付出努力的期间；费用归属于其带来收益的期间。

### 资本化与折旧 Capitalization & Depreciation
某些支出不作为当期费用，而是作为资产，在其有用寿命内逐步折旧。

### 权责发生制 Accrual & Deferral
将收入和费用分配到适当的时期，即使现金尚未收付。
- **应计 (Accrual)**：已赚取但未收到的收入 / 已发生但未支付的费用
- **递延 (Deferral)**：已收到但未赚取的收入 / 已支付但未消耗的费用

### 收入确认 Revenue Recognition
决定何时将交易记入收入或费用账户。这是财务舞弊最常见的操纵点之一。

## 存货计价方法

| 方法 | 特点 | 对报表的影响 |
|------|------|-------------|
| FIFO (先进先出) | 先买的先卖 | 利润表更准确（成本反映较早价格），资产负债表更准确（存货接近当前成本） |
| LIFO (后进先出) | 后买的先卖 | 在通胀时利润较低（税盾效应），存货可能低估 |
| 加权平均 | 平均成本 | 折中方案 |
| 重置成本 | 当前购买价格 | 最接近市场现实 |

**权衡**：利润表准确性 vs 资产负债表准确性不可兼得。

## 公允价值 vs 历史成本

| | 公允价值 | 历史成本 |
|---|---|---|
| 优势 | 更及时、更相关 | 更客观、更可靠 |
| 劣势 | 主观性强、易被操纵 | 可能严重偏离真实价值 |

## 制度框架

### 标准与机构

| 机构 | 角色 |
|------|------|
| **GAAP** | 美国通用会计准则（主流规范） |
| **GAAS** | 通用审计准则 |
| **FASB** (1973年成立) | 制定 GAAP |
| **AICPA** | 美国注册会计师协会 |
| **PCAOB** (2002年成立) | 监管上市公司审计师 |
| **SEC** | Regulation S-K, S-X；仅适用于不到1%的美国公司 |
| **四大** | Deloitte, EY, KPMG, PwC |

### SEC 关键文件

| 文件 | 内容 |
|------|------|
| **10-K** | 年报，经审计 |
| **10-Q** | 季报 |
| **DEF-14K** | 委托声明书 (proxy statement) |
| **8-K** | 重大事件报告 |

### 审计意见等级
- **审计报告 (Audited)**：无保留意见 — 合理保证 (reasonable assurance)
- **审阅 (Review)**：有限保证 (limited assurance)
- **编制 (Compilation)**：不提供准确性保证

## 财务比率

### 盈利能力
- **毛利率** = 毛利 / 销售收入
- **营业利润率 (Operating Margin)** = 营业利润 / 销售收入
- **ROA (资产回报率)** = 净利润 / 总资产
- **ROE (权益回报率)** = 净利润 / 股东权益

### 市场估值
- **EPS (每股收益)** = 净利润 / 流通股数
- **P/E (市盈率)** = 平均股价 / EPS
- **每股账面价值** = 股东权益 / 流通股数

### 管理效率
- 应收账款周转率、存货周转率 (COGS / 存货)、利息成本

### 流动性
- **流动比率 (Current Ratio)** = 流动资产 / 流动负债

## 律师实务场景

### 尽职调查
- 阅读三大报表找出异常
- 关注收入确认方式（是否激进？）
- 比较多年数据趋势
- 检查表外项目

### 财务舞弊识别
**Sterling Homex 案例**：建设公司通过操纵收入确认虚增利润，1972年破产，高管入狱。

**红旗信号**：
- 收入增长远超行业平均但现金流为负
- 频繁变更会计政策
- 异常的关联交易
- 审计师变更

### 诉讼中的财务分析
- 用财务比率评估对方的支付能力
- 在损害赔偿计算中理解对方的利润结构
- 在破产程序中评估资产充足性
