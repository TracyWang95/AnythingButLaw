# 金融 Finance

> 几乎所有法律争议最终都涉及一个金额。理解货币的时间价值、折现率、估值方法和资本市场，是律师量化损害赔偿、评估交易价值、识别金融欺诈的基础。

## 企业理论 Theory of the Firm

理解现代公司为什么是现在这个样子：

| 学者 | 贡献 |
|------|------|
| **Coase** | 《企业的性质》— 交易成本解释了企业存在的原因 |
| **Berle & Means** | 所有权与控制权的分离 — 公司治理问题的根源 |
| **Modigliani & Miller** | 资本结构理论 — 资本成本 |
| **Jensen & Meckling** | 管理者行为 — 代理成本 |

## 货币的时间价值 Time Value of Money

**核心原理**：今天的一块钱比未来的一块钱更值钱。

- 越遥远的未来，现值越低
- 折现率越高，现值越低

### 单利与复利

- **单利 (Simple Interest)**：只对本金计息
- **复利 (Compound Interest)**：对本金和未付利息计息 — 复利的力量是巨大的

### 72法则 Rule of 72
投资翻倍所需年数 ≈ 72 / 利率
- 6%利率 → 约12年翻倍
- 12%利率 → 约6年翻倍

### 核心公式

**现值 (Present Value)**：
```
PV = FV / (1 + r)^n
```

**终值 (Future Value)**：
```
FV = PV × (1 + r)^n
```

**年金 (Annuity)**：等额定期支付的系列（普通年金：期末支付）

**内部收益率 (IRR)**：使净现值 (NPV) = 0 的折现率

### 折现率的选择

折现率取决于 **付款方的风险特征**：
- 联邦政府：最低（违约概率最小）
- 大型蓝筹公司（如 Microsoft）：较低
- 高科技公司：较高
- 个人：通常最高

还需考虑：自身资金成本、风险规避程度。

**律师关键洞察**：对方提出的折现率是否合理？折现率每变化1个百分点，在长期现金流中可能导致数百万的差异。

## 估值方法

### 1. 可比交易法
最简单 — 找到涉及基本相似资产的近期交易，以其价格为参考。

### 2. 基于财务报表的市场估值
- **账面价值 (Book Value)**：资产负债表上的权益值
- **资本化收益 (Capitalized Earnings)**：用收益除以适当的资本化率
- **价格乘数 (Price Multiples)**：P/E比率、市净率 (Market-to-Book) 等

### 3. 折现现金流 (DCF) 分析

最严谨但也最敏感于假设的方法：

1. **预测预期现金流** — 不是利润，是现金流
2. **选择适当的折现率** — 通常基于 CAPM（高Beta → 高折现率）
3. **计算现值**
4. **判断**：如果投资成本 < 预测现金流的现值 → 正 NPV → 值得投资

**律师注意**：DCF分析的结果极度依赖于假设（增长率、折现率、预测期限）。质疑这些假设是律师的核心工作。

## 有效市场假说 Efficient Market Hypothesis (EMH)

大型股票市场会迅速将重要公开信息纳入价格。

**含义**：
- 投资者无法利用某些类型的信息获得超额收益
- 不是说价格总是"正确的"或反映长期价值

**法律连接**：
- **内幕交易**：基于"市场欺诈理论 (fraud on markets theory)" — 如果市场有效，内幕交易就是利用市场尚未知道的信息获利
- **证券欺诈**：虚假信息导致市场价格偏离

## 风险与回报 Risk & Return

**风险**：回报的不确定性，衡量回报的预期变异程度
**回报**：一段时期内的总经济收益（定期支付 + 增值）

### 历史实证
高回报的投资在历史上也产生了更大的波动：
```
政府债券 < 公司债券 < 大公司股票 < 小型/创业公司股票
风险：低 ————————————————————————————→ 高
回报：低 ————————————————————————————→ 高
```

**律师诊断工具**：当某个投资承诺高回报但声称低风险时 — 这是一个危险信号（可能是欺诈）。

## 分散投资 Diversification

"不要把所有鸡蛋放在一个篮子里。"

- 约30个不同投资即可大幅降低个别公司/行业的特有风险
- **关键问题**：通过分散投资能消除多少风险？（系统性风险无法消除）

## CAPM 与 Beta

**资本资产定价模型 (CAPM)**：某只股票的回报取决于整体市场的变动。

**Beta (β)**：
- β = 1.0：与市场同步
- β = 1.5：市场涨10%时涨15%，跌10%时跌15%
- β < 1.0：波动小于市场

**用途**：确定权益资本成本和适当的折现率。

## 证券基础

**证券 (Securities)**：具有货币价值、可交易的金融工具。
- **权益类 (Equity)**：股票 — 所有权
- **债务类 (Debt)**：债券、信用债 (debentures) — 借贷关系

## 律师实务场景

### 损害赔偿计算
- 用现值公式将未来收入损失折算为今天的金额
- 选择折现率时考虑原告的风险特征
- 一次性支付 vs 分期支付的比较

### 企业估值（并购/离婚财产分割/合伙纠纷）
**Eskimo Pie 案例**：使用了多种估值方法 — 账面价值、市净率、净利润乘数、DCF — 并与可比公司（Ben & Jerry's, Dreyer's, Hershey 等）进行比较。最终被 Nestle 收购。权益资本成本 15.1%。

### Powerball 问题
一次性领取 vs 分25年领取的年金 — 用时间线分析和现值计算判断哪个更划算。

### 家庭财产分割
Eve Adamson 问题：不同家庭成员有不同需求时的公平分配 — 涉及隐性转移、逆向选择和道德风险。

### 投资欺诈识别
- 承诺回报是否符合风险-回报关系？
- 是否存在有效市场假说无法解释的持续超额回报？
- 折现率假设是否合理？
