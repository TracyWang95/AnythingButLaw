# 博弈论与信息 Game Theory & Information

> 律师和客户往往无法控制自己的命运 — 结果取决于对手的行动。博弈论提供了一套语言和工具来描述和预测战略行为。

## 博弈论基础

**博弈论之父**：John von Neumann
**纳什均衡**：John Nash（诺贝尔奖，《美丽心灵》）

### 博弈的要素

| 要素 | 说明 | 法律举例 |
|------|------|----------|
| 玩家 (Players) | 2个或以上的决策者 | 原告、被告、监管者 |
| 行动 (Actions) | 每个玩家可选择的动作 | 提起诉讼、和解、上诉 |
| 策略 (Strategy) | 在每个可能节点的行动计划 | 完整的诉讼策略方案 |
| 信息 (Information) | 每个玩家掌握的信息 | 证据、对方的底线 |
| 收益 (Payoffs) | 各结果下的得失 | 赔偿金、诉讼费、名誉 |
| 时序 (Timing) | 同时或依次行动 | 密封投标 vs 公开谈判 |

### 博弈表 Game Tables
- 一方（如原告）在侧面，另一方（如被告）在顶部
- 收益：侧面玩家的数字写在前面，顶部玩家的数字写在后面
- 注意正负号，不要忘记诉讼成本

## 解题概念 Solution Concepts

### 优势策略 Dominant Strategy
无论对手怎么做，某个策略都是最优的。如果存在优势策略，直接选它。

**迭代删除 (iterated dominance)**：先删除一方的被支配策略，在缩小的博弈中继续删除，逐步简化。

**准优势策略 (quasi-dominant strategy)**：不是严格优势，但在大多数情况下更好。

### 纳什均衡 Nash Equilibrium
每个玩家在给定对手策略的情况下，都不愿意单方面改变自己的策略。

- 可能存在多个纳什均衡
- **焦点 (focal point)**：当存在多个均衡时，某个"自然"的选择点

### 子博弈完美 Subgame Perfect
在博弈的每个阶段（子博弈）都是理性的。用来排除不可信的威胁。

### 囚徒困境 Prisoners' Dilemma
两个玩家如果合作，双方都更好；但每个人的优势策略是背叛。

**关键洞察**：
- 光靠沟通不够 — 还需要执行机制或声誉约束
- 在法律实务中：为什么需要合同（而不是口头承诺）、为什么需要监管

## 威胁的可信性

### 空洞威胁 Empty Threats (Non-credible Threats)
理性玩家不会执行的威胁，因为执行对自己不利。

**例**：如果一家公司威胁要打价格战来阻止新竞争者进入，但打价格战会让自己亏损更多 — 这就是空洞威胁。

### 可信威胁 Credible Threats
执行威胁符合威胁方的利益，或者威胁方已经做出不可逆的承诺。

**律师实务**：识别对方的威胁是否可信，是谈判中的关键技能。

### 沉没成本 Sunk Cost
已经发生、无法收回的成本。理性决策应忽略沉没成本 — 但人们经常做不到。

## 信息不对称

### 道德风险 Moral Hazard
一方的行为对另一方不利，但因为信息不对称而无法被充分监控。

**起源**：保险业 — 投保后人们减少预防措施，因为损失有人兜底。

**法律场景**：
- 律师按小时收费 → 可能过度工作（客户无法完全监控）
- 代理人有保险 → 可能降低注意义务
- 公司高管 → 可能追求个人利益而非股东利益

**解决方案**：合同条款、监控、激励设计（对齐利益）

### 逆向选择 Adverse Selection
交易前的信息不对称。知情方利用信息优势，导致市场扭曲。

**经典例子**：
- 二手车市场（"柠檬"问题）
- 保险市场 — 高风险者更倾向于购买保险
- 保险公司的对策：限制基础保障范围，让高风险者自己购买附加保险

**法律场景**：
- 并购中的尽调 → 卖方知道的比买方多
- 赦免/自首项目 → 什么样的人会来自首？

## 谈判与保留价格

### 保留价格 Reservation Price
一方愿意接受的最低（或最高）价格。

**谈判空间** = 买方保留价格 - 卖方保留价格（如果 > 0，交易有可能达成）

**关键**：
- 谈判中的行为可以帮助揭示对方的概率判断
- 双方都可以从设计良好的激励中受益（参见合同设计）
- 非财务收益（声誉、先例价值、情绪）也影响保留价格

## 法律应用场景速查

| 场景 | 主要博弈论工具 |
|------|---------------|
| 诉讼和解谈判 | 博弈表、纳什均衡、保留价格 |
| 合同谈判 | 囚徒困境、威胁可信性 |
| 反垄断（价格战、市场进入） | 子博弈完美、空洞威胁 |
| 保险合同设计 | 道德风险、逆向选择 |
| 公司治理 | 道德风险（代理问题） |
| 监管设计（赦免项目等） | 逆向选择、激励相容 |
| 并购尽调 | 逆向选择、信息揭示 |
