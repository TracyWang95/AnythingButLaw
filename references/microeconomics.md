# 微观经济学 Microeconomics

> 微观经济学给律师提供了一套理解市场如何运作（以及如何失灵）的框架。反垄断、消费者保护、环境法、税法 — 几乎所有领域都能从供需分析中获益。

## 竞争市场 Competitive Markets

### 特征
- 每个参与者都是"小角色"，接受市场价格
- 价格和总交易量由供给和需求共同决定
- 每个企业是"价格接受者 (price taker)"而非"价格制定者 (price maker)"
- 需要：充分竞争 + 自由进入

### 需求曲线 Demand Curve
- 价格上升 → 总需求下降（向下倾斜）
- 横轴：数量；纵轴：价格
- **移动因素**：
  - 个人收入变化（正向）
  - 替代品价格变化（正向）— **替代品 (substitutes)**：可互相替换的商品
  - 互补品价格变化（反向）— **互补品 (complements)**：需要搭配使用的商品

### 供给曲线 Supply Curve
- 价格上升 → 企业愿意供给更多（向上倾斜）
- 短期供给曲线比长期更陡峭（短期内产能受限）
- **移动因素**：制造成本变化、自然灾害等

### 均衡 Equilibrium
供给曲线和需求曲线的交点 — 市场出清价格和数量。

## 价格弹性 Price Elasticity of Demand

衡量需求量对价格变化的敏感度。

```
弹性 = |需求量变化百分比 / 价格变化百分比|
```

| 弹性 | 含义 | 涨价后果 |
|------|------|----------|
| > 1（弹性大） | 需求对价格敏感 | 收入减少 |
| < 1（弹性小） | 需求对价格不敏感 | 收入增加 |
| = 1 | 恰好平衡 | 收入不变 |

**其他弹性**：收入弹性、交叉价格弹性

**律师应用**：在反垄断中评估垄断者涨价的影响；在损害赔偿中评估替代品的可得性。

## 消费者剩余与生产者剩余

- **消费者剩余 (Consumer Surplus)**：消费者愿意支付的价格 - 实际支付的价格（需求曲线与价格线之间的面积）
- **生产者剩余 (Producer Surplus/Profit)**：实际收到的价格 - 生产成本（价格线与供给曲线之间的面积）
- **总剩余 = 价值 - 生产成本 (v - c)**
- 在竞争均衡数量处，总剩余最大化

**律师意义**：损害赔偿通常是"如果没有侵权行为，你本来会有多少剩余？"

## 政府干预

### 价格下限 Price Floor（最低价格）
供给 > 需求 → 需要分配机制（如最低工资）

### 价格上限 Price Ceiling（最高价格）
需求 > 供给 → 需要分配机制（如租金管制）

### 补贴 Subsidies
- 需求侧补贴：增加消费者购买力
- 供给侧补贴：降低生产者成本

### 商品税 Commodity Taxes
减少需求或增加供给成本。

**律师注意**：每种干预都会产生效率损失（无谓损失），但可能基于公平考量是合理的。

## 垄断 Monopoly

### 为什么垄断是问题？
垄断者关注 **边际收益 (marginal revenue)**，而非价格。它会限制产量、抬高价格，产生 **无谓损失 (deadweight loss)** — 总社会剩余减少。

### 法律应用
- **反垄断/并购**：合并后是否会形成垄断？
- **价格歧视 (price discrimination)**：对不同消费者收取不同价格
- **自然垄断 (natural monopoly)**：某些行业只有一家企业效率最高（如公用事业）→ 需要监管

## 外部性 Externalities

一方的行为影响了不参与交易的第三方。

**负外部性**：污染、噪音
**正外部性**：教育、疫苗

### 解决方案

| 方案 | 特点 |
|------|------|
| 私人谈判 | Coase 定理的理想情况 |
| 创建市场 | 碳排放交易 |
| 法律干预 | 监管、产权、禁令、侵权责任、税收 |

### 科斯定理 Coase Theorem
在交易成本足够低的情况下，无论产权如何分配，私人谈判都可以达到社会最优结果。

**局限**：现实中交易成本很少足够低 — 所以需要法律。

## 公共物品 Public Goods

**两个特征**：
- **非排他性 (nonexcludable)**：无法阻止他人使用
- **非竞争性 (nonrival)**：一个人使用不减少他人使用

**问题**：搭便车 → 私人供给不足 → 需要政府提供

## 福利经济学 Welfare Economics

### 分析框架
- **剩余**：效率度量
- **功利主义 (Utilitarianism)**：最大化总效用（Bentham, Sidgwick）
- **最大最小原则 (Maximin)**：最大化最差处境者的福利
- **矫正正义 (Corrective Justice)**：受到不当伤害者应获得赔偿

### 关键权衡
- 矫正正义与效率之间的冲突
- 再分配不是无成本的（收入税会削弱工作积极性）
- 产出减少 vs 再分配带来的社会福利收益

| 法律领域 | 核心经济学概念 |

| 反垄断 | 垄断、无谓损失、市场定义、价格歧视 |

| 公司法 | 代理问题、外部性 |

| 破产 | 债权人优先级、效率 |
| 产品责任/侵权 | 外部性、科斯定理 |
| 知识产权 | 公共物品、搭便车 |
| 劳动法/歧视法 | 供需、工资弹性 |
| 环境法 | 外部性、科斯定理、排放交易 |
| 行政法 | 市场失灵、监管正当性 |
