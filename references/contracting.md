# 合同设计 Contracting

> "像合同设计师一样思考" — 律师不只是审查合同语言，更是设计一套激励系统。好的合同让双方的利益对齐，坏的合同制造道德风险。

## 核心理念：做大蛋糕

合同设计的目标不是帮一方多拿，而是 **增加合同安排的总价值**。

- 加入能增加净价值的条款，移除减少净价值的条款
- 评估每个条款创造的有意和无意的激励
- 对价不一定是钱 — 也可以是声誉、优先权、排他权等
- "贪心但别太贪" — 如果交易谈崩了，谁也享受不到蛋糕

## 六种合同类型

### 1. 固定费用合同 Flat-Fee
支付约定的固定金额。

| 优势 | 风险 |
|------|------|
| 买方成本确定 | 承包方有动机偷工减料用最差材料 |
| 简单明了 | 变更需重新谈判，成本高且可能被要挟 (holdups) |
| 通常更便宜 | 买方难以验证质量 |

**风险承担**：承包方承担全部财务风险。

### 2. 成本加成合同 Cost-Plus
买方支付实际成本 + 一定利润。

| 优势 | 风险 |
|------|------|
| 不需重新谈判 | 承包方无动力控制成本 |
| 质量有保障 | 可能"镀金"— 过度追求质量 |
| | 买方难以验证成本真实性 |

**风险承担**：买方承担全部财务风险。

### 3. 绩效合同 Performance-Based (Output-Based)
报酬取决于按特定标准衡量的产出。

**例**：律师按胜诉金额的百分比收费（风险代理）。

**优势**：激励代理人尽力表现
**风险**：将风险转移给代理人；产出指标可能被扭曲

### 4. 投入合同 Input-Based
报酬与投入（如时间）挂钩。

**例**：律师按小时收费。

**优势**：比绩效合同更易管理
**风险**：忽略了难以观察的绩效维度；代理人可能磨洋工

### 5. 定额合同 Fixed-Fee
支付固定金额，无论投入或产出。

**风险**：代理人完全没有表现激励。

### 6. 混合合同 Hybrid
包含奖金或罚款的组合结构。

**例**：基础时薪 + 胜诉奖金；固定价格 + 延期罚款。

## 合同设计的核心维度

### 激励 Incentives

**类型**：财务激励、声誉激励、政治激励、排他性

**设计要点**：
- 激励的强度和上限 — 太弱没效果，太强可能导致极端行为
- 质量关注 — 只激励数量，质量可能下降
- **"尽最大努力" (best efforts) 通常不够** — 因为无法验证，形同虚设

### 可验证性 Verifiability

最关键的问题：你能证明对方做了（或没做）什么吗？

- 固定费用合同中买方难以验证质量
- 成本加成合同中买方难以验证成本
- 绩效合同需要可衡量的产出指标
- 不可验证的条款实际上不可执行

### 风险分配 Risk Allocation

- 谁承担风险？谁更适合承担风险？
- 绩效合同：代理人承担与利润分成比例对应的风险
- 投入合同和定额合同：通常不对代理人施加风险
- 当委托人相对于代理人不太厌恶风险时，双方都倾向投入合同或定额合同

### 或有条款 Contingent Provisions

预判可能出现的问题：延误、事故、罢工、隐蔽缺陷。

**原则**：只在问题发生的可能性和重要性足以证明额外起草成本合理时，才添加或有条款。利用合同法知识预估法庭/仲裁庭的解释倾向。

## 违约救济

### 期望损害赔偿 Expectation Damages
默认救济 — 让受害方处于合同被履行时的状态。

### 约定损害赔偿 Liquidated Damages
双方预先约定的赔偿金额。如果金额过高，可能被认定为罚金 (penalty) 而不可执行。

### 替代机制
- **调解 (mediation)**：第三方协助谈判
- **仲裁 (arbitration)**：第三方做出有约束力的裁决

## 具体场景中的合同保护

### 不动产合同
具体要求、检查权、保证金、罚款、转租权、运输中损坏、披露义务、保证、赔偿条款。

### 借贷合同
- **贷方视角**：抵押担保（如按揭）、限制性条款 (covenants)、支出限制、保险要求
- **借方视角**：提前还款权、再融资权、合理限制换取更低利率

## 常见错误

1. 保留减少净价值的条款（因为"一直都这样写"）
2. 固定费用/投入/定额合同中忽略质量问题
3. 成本加成/投入合同中不控制价格上限
4. 错误分配风险
5. 不考虑争议解决和退出机制
6. 谈判释放条款 (release) 但没有保护后续索赔
7. 机械套用合同类型，不根据具体情况调整

## 合同类型选择速查

| 你最担心什么？ | 推荐类型 | 原因 |
|---|---|---|
| 成本确定性 | 固定费用 | 承包方承担超支风险 |
| 质量保证 | 成本加成 | 承包方没有偷工减料的动机 |
| 代理人努力程度 | 绩效合同 | 直接对齐利益 |
| 项目范围不确定 | 成本加成/投入 | 不需重新谈判 |
| 双方信任度低 | 混合合同 + 仲裁 | 奖惩并用 + 低成本争议解决 |
