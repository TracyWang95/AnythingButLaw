# 统计分析 Statistical Analysis

> 律师不需要自己跑统计，但需要知道对方的统计证据是否站得住脚。在 Daubert 时代，理解统计推断是评估和挑战专家证词的必备技能。

## 描述统计 Descriptive Statistics

### 集中趋势 Measures of Central Tendency

| 指标 | 含义 | 何时用 |
|------|------|--------|
| **均值 (Mean)** | 算术平均数 | 数据分布大致对称时 |
| **中位数 (Median)** | 排序后的中间值 | 数据有偏斜或异常值时（更稳健） |
| **众数 (Mode)** | 出现次数最多的值 | 分类数据 |

**关键**：当数据变异性大时，均值和中位数作为数据摘要的价值降低。

### 变异性 Measures of Variability

| 指标 | 公式 | 说明 |
|------|------|------|
| **极差 (Range)** | 最大值 - 最小值 | 最简单，但受异常值影响大 |
| **方差 (Variance)** | 各值与均值之差的平方的平均 | 总体方差除以 n；样本方差除以 n-1 |
| **标准差 (Standard Deviation)** | 方差的平方根 | 与原数据同单位，最常用 |

**Excel 函数**：AVERAGE, MEDIAN, MODE.SNGL, MODE.MULT, MAX, MIN, VAR.P, VAR.S, STDEV.P, STDEV.S

## 数据展示

### 常见图表
- **柱状图 (Bar Graph)**：通常比表格更直观有效
- **直方图 (Histogram)**：连续数据的柱状图
- **散点图 (Dot Plot)**：显示数据分布

### 常见陷阱
- **Y轴不从零开始**：最常见的误导手法 — 让微小差异看起来巨大
- 忽略原始数据直接看汇总统计
- 调查设计的效度 (validity) 问题
- 异常值和数据缺口

## 正态分布 Normal Distribution

理想的对称钟形曲线。越集中在均值附近，曲线越"尖"。

### Z分数 Z-Score
描述一个值离均值有多少个标准差。

```
Z = (观测值 - 均值) / 标准差
```

**Z表**：给出正态分布中z分数小于给定值的百分比。

**Excel**：NORM.S.DIST, NORM.S.INV, NORM.INV, NORM.DIST

## 推断统计 Inferential Statistics

### 样本与总体
- **总体 (Population)**：你想研究的完整群体
- **样本 (Sample)**：从总体中抽取的一部分
- **抽样方法 (Sampling Method)**：样本的选取方式

### 有效推断的前提
- 适当的抽样技术（随机性）
- 足够的样本量
- 效度 (validity)：测量的是你想测量的东西

### 常见偏差
- **动机偏差 (Motivational Bias)**：受访者迎合期望
- **默认偏差 (Acquiescence Bias)**：倾向于同意
- **认知困难**：问题太复杂
- **模糊性**：问题含义不清

## 假设检验 Hypothesis Testing

### 流程
1. 提出 **零假设 (null hypothesis)** — 通常是"没有差异"或"没有效果"
2. 随机抽样
3. 计算检验统计量
4. 解释结果

### Z检验公式

**均值检验**：
```
Z = (样本均值 - 总体均值) / (总体标准差 / √样本量)
```

**比例检验**：
```
Z = (样本比例 - 总体比例 p) / √(p(1-p) / 样本量)
```

### 统计显著性 Statistical Significance
差异大到仅有5%或更小的概率由偶然产生。

**临界值**：z < -1.96 或 z > 1.96（双尾，5%显著性水平）

### 两类错误

| | 实际上有效果 | 实际上没效果 |
|---|---|---|
| **结论：有效果** | ✓ 正确 | **I类错误** (False Positive) |
| **结论：无效果** | **II类错误** (False Negative) | ✓ 正确 |

**法律类比**：
- I类错误 ≈ 误判有罪（冤枉无辜）
- II类错误 ≈ 放走罪犯

### 常见陷阱

- **"抽屉问题" (File Drawer Problem)**：不显著的研究被藏起来不发表 → 发表偏差
- **"癌症簇问题" (Cancer Cluster Problem)**：在没有全国背景数据的情况下，单看某个县的癌症率
- **统计显著性 ≠ 实际意义**：极大的样本量可以让微小的差异变得"显著"

## 中心极限定理 Central Limit Theorem

从总体中反复抽取大小为 n（至少30）的随机样本，样本均值的分布近似正态分布：
- 样本均值的均值 = 总体均值
- 样本均值的标准差 = 总体标准差 / √n
- 总体比例的标准差 = √(p(1-p))

## 置信区间 Confidence Intervals

用样本估计总体参数：

```
置信区间 = 样本均值 ± (z分数 × 样本均值的标准差)
```

95%置信区间：z = 1.96

**含义**：如果我们重复抽样100次，约95次的区间会包含真实的总体参数。

## Daubert 标准

最高法院对"垃圾科学"进入法庭的回应。法官作为科学证据的"守门人"。

**律师应关注的统计证据质量问题**：
- 抽样是否随机？样本量是否足够？
- 零假设是否合理？
- 显著性水平是否适当？
- 是否存在发表偏差或数据挖掘？
- 统计显著性是否转化为实际意义？
- 效果量 (effect size) 有多大？
