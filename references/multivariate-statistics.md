# 多元统计 Multivariate Statistics

> 歧视诉讼中，被告说"我们不是因为性别，是因为经验不同"。你怎么检验这个说法？答案是多元回归 — 控制其他变量后，看性别是否仍然显著影响薪酬。这是律师在现代诉讼中最常遇到的统计工具。

## 散点图 Scatterplots

探索两个变量关系的第一步。

**绘制方法**：
1. X轴标注解释变量，选择合适刻度
2. Y轴标注响应变量，选择合适刻度
3. 在交叉点标记每个观测值

**用途**：揭示变量之间的关系类型（线性、非线性、无关系）。

## 皮尔逊相关系数 Pearson Correlation Coefficient (r)

衡量两个变量之间 **线性关系** 的方向和强度。

| |r| 范围 | 强度 |
|---|---|
| < 0.1 | 不相关 |
| 0.10 - 0.30 | 弱相关 |
| 0.30 - 0.50 | 中等相关 |
| ≥ 0.50 | 强相关 |

**计算**：X的z分数 × Y的z分数 → 求和 → 除以 (n-1)

**Excel**：PEARSON 或 CORREL

**注意**：异常值 (outliers) 会严重扭曲相关系数。

## 相关 ≠ 因果

这是统计学中最重要、也是律师最需要铭记的原则。

### 相关不意味着因果
两个变量一起变动，不代表一个导致了另一个。

### 因果也不一定表现为相关
例：动机与表现可能呈倒U型关系 — 适度动机最优，过高过低都差。线性相关系数捕捉不到这种非线性关系。

### 混淆因素

**共同反应 (Common Response)**：
X和Y都因为第三个变量Z的变化而变化，但X和Y之间没有因果关系。
- 例：冰淇淋销量和溺水事故都在夏天增加 — 共同原因是气温

**混杂 (Confounding)**：
X对Y的影响被另一个潜伏变量的影响所混淆。
- 例：教育年限与收入相关，但可能被家庭背景混淆

## 简单线性回归 Simple Linear Regression

### 回归方程
```
Y = a + bX
```
- a = 截距 (intercept)
- b = 斜率 (slope) — X每变化一个单位，Y的预期变化

### 特点
- 对所有具有相同X值的个体，预测相同的Y值
- 只适用于近似线性的关系
- 误差几乎不可避免 — 响应变量的变异是常态

**Excel**：添加趋势线并显示方程和R²；INTERCEPT 和 SLOPE 函数

## R² 决定系数 Coefficient of Determination

```
R² = (TSS - SSR) / TSS
```
- **TSS (Total Sum of Squares)**：响应变量的总变异
- **SSR (Sum of Squared Residuals)**：模型无法解释的变异

**含义**：模型解释了响应变量变异的多大比例。
- R² = 0.80 → 模型解释了80%的变异
- R² = 0.20 → 模型只解释了20%的变异

**Excel**：RSQ 函数

## 多元回归 Multiple Regression

### 方程
```
Y = a + b₁X₁ + b₂X₂ + ... + bₙXₙ
```

**核心优势**：利用额外信息提高预测准确性；**控制其他变量**后隔离某个特定变量的影响。

### 法律应用

**歧视诉讼中的核心工具** — 例如 **Bazemore v. Friday, 478 U.S. 385 (1986)**

典型分析：
- Y = 薪酬
- X₁ = 性别（虚拟变量）
- X₂ = 工作经验
- X₃ = 教育程度
- X₄ = 部门

如果控制经验、教育、部门后，性别的回归系数仍然显著 → 存在薪酬歧视的统计证据。

### t检验
如果某个回归系数的t检验未通过所选显著性水平 → 无法得出该解释变量在控制其他变量后仍有统计显著影响的结论。

## 虚拟变量 Dummy Variables

将分类变量编码为数值的方法。

**例**：性别 → 女性 = 1，男性 = 0（赋值是任意的）

回归系数的含义：在控制所有其他变量后，女性比男性平均薪酬高/低多少（取决于系数正负）。

## 偏离良好实践的常见问题

### 1. 遗漏重要解释变量
**最常见也最致命的问题。**

- 如果遗漏了一个既影响响应变量又与已包含变量相关的变量 → 已包含变量的系数会有偏
- **Sears 佣金销售案例**：忽略了"对该工作的兴趣"这个变量，导致歧视结论不可靠

**律师策略**：质问对方专家"你为什么没有控制XX变量？"

### 2. 包含无关变量
如果测试太多变量，某些会仅因抽样误差而显得"显著"（数据挖掘/p-hacking）。

### 3. 多重共线性 Multicollinearity
某个解释变量与其他解释变量高度相关 → 系数估计不稳定、不可靠。

**例**：同时包含"工龄"和"年龄" — 它们高度相关，难以分离各自的影响。

### 4. 双向因果 Two-Way Causation
解释变量影响响应变量，同时响应变量也反过来影响解释变量。

**经典例子**：警力 (police) 与犯罪率 (crime)
- 更多警察 → 更少犯罪？
- 更多犯罪 → 部署更多警察？
- 简单回归可能发现"警察越多，犯罪越多" — 但这是因果方向错误

## 律师实务场景速查

| 场景 | 关键关注点 |
|------|-----------|
| 就业歧视诉讼 | 是否控制了所有相关变量？虚拟变量是否正确编码？ |
| 薪酬公平分析 | R²是多少？遗漏变量偏误？ |
| 环境/健康诉讼 | 因果推断的可靠性？混淆因素？ |
| 反垄断/市场分析 | 双向因果？需求和价格的内生性？ |
| 专家证词挑战 | 变量选择的合理性、多重共线性、模型假设检验 |

## 挑战回归分析的律师清单

1. **变量选择**：是否遗漏了重要变量？包含了无关变量？
2. **因果方向**：是否存在双向因果？
3. **线性假设**：关系是否真的是线性的？
4. **异常值**：是否有异常值扭曲了结果？
5. **多重共线性**：解释变量之间是否高度相关？
6. **样本选择**：样本是否具有代表性？
7. **统计显著性 vs 实际意义**：系数虽然显著，但效果量够大吗？
8. **R²**：模型解释力如何？未解释的80%去哪了？
